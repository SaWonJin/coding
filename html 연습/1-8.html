ol태그에 타입 속성을 사용하면 영문자나 로마숫자도 나타낼 수 있다.

  1. type = "1" / 숫자(기본값)
  2. type = "a" / 영문 소문자
  3. type = "A" / 영문 대문자
  4. type = "i" / 로마 숫자 소문자
  5. type = "I" / 로마 숫자 대문자

start속성을 사용하면 첫번째가 아닌 속성에 적은 숫자부터 시작
