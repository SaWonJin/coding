폼을 만드는 form 태그

form 태그의 속성
종류 설명
method 사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄지 지정
사용할 수 있는 속성값은 get과 post이다.
get : 주소 표시중에 사용자가 입력한 내용이 드러남
post : 입력한 내용의 길이에 제한받지 않고 사용자 내용도 드러나지 않음
name 폼의 이름을 지정한다.
action 내용을 처리해 줄 서버 프로그램을 지정한다.
target action속성에서 지정한 파일을 다른 위치에서 열도록 한다.

자동완성 기능을 나타내는 autocomplete

끄고 싶을 땐 autocomplete="off" 하면 된다.

폼 요소에 레이블을 붙이는 label 태그

label이란 입력란 가까이에 아이디나 비밀번호처럼 붙여 놓은 텍스트를 말한다.

label과 input 요소를 따로 사용할 경우 id를 정확하게 맞춰줘야한다.


input 태그의 type 속성 한눈에 살펴보기

텍스트와 비밀번호

type="text" / type="password"

text와 password에서 사용하는 속성
종류 설명
size 텍스트와 비밀번호 필드의 길이를 지정
화면에 몇 글자가 보이도록 할 것인지를 지정한다.
value 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여 주는 내용
maxlength 텍스트 필드와 비밀번호 필드에 입력할 수 있는 최대 문자수를 지정한다.

search, url, email, tel

type="search"를 사용하면 웹 브라우저에서는 검색을 위한 텍스트 필드로 인식한다.

type="url" 웹 주소를 입력하는 필드

type="email" 이메일 주소를 입력하는 필드

type="tel" 전화번호를 나타내는 필드

세분화된 필드는 pc에선 별 차이가 없어 보이지만 모바일 기기에서 확인하면 키보드 배열이 바뀐다.

checkbox, radio

여러 항목 중에서 원하는 항목을 선택할 때 사용하는 폼 요소

1개만 선택하려면 라디오버튼

2개 이상 선택하려면 체크박스 사용

종류 설명
value 숫자나 영어로 서버에게 넘겨줄 값을 지정함
checked 여러 항목 중에서 기본으로 선택해 놓고 싶은 항목에 사용

number, range

type="number"를 사용하면 스핀박스가 나타나면서 숫자를 선택할 수 있다.

type="range"는 슬라이드 막대를 움직여 숫자를 입력할 수 있다.


숫자 입력 필드에서 사용하는 속성
속성 설명
min 필드에 입력할 수 있는 최솟값 지정
max 필드에 입력할 수 있는 최댓값 지정
step 숫자 간격을 지정할 수 있다. 기본값은 1
value 필드에 표시할 초기값값

date, month, week

date는 연도,월,일 표시

month는 연도, 월 표시

week는 주를 선택해서 입력

time, datetime, datetime-local

시간을 지정할 땐 time사용

날짜와 시간을 같이 지정할 땐 datetime, datetime-local 사용

submit, reset

전송 버튼 submit
submit을 누르면 form태그의 action 속성에 지정한 곳으로 넘겨진다.

삭제 버튼 reset

image

submit과 같은 기능을 하며 이미지 버튼을 나타낸다.

button

기본 버튼을 나타내며 따로 기능은 없어서 자바스크립트를 활용해 넣어야한다.

file

화면에 [파일선택] 이나 [찾아보기] 버튼 등이 표시된다.

hidden

화면에선 보이지 않지만 사용자가 입력을 마치면 폼과 함께 서버로 보내야 하는 요소들

사용자의 아이디정보 등이 넘어갈 때 사용한다.
